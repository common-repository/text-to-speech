== Description ==

Javascript tool for transforming the text into speech.

Based on eSpeak code made by Jonathan Duddington.

Relevant links:
http://en.wikipedia.org/wiki/ESpeak
http://espeak.sourceforge.net/
http://sourceforge.net/users/jonsd
http://jonathan-duddington.mac.informer.com/
	


== Installation ==

1. Extract and upload `text-to-speech.zip` to the `/wp-content/plugins/` directory;
2. Activate the plugin through the 'Plugins' menu in WordPress;
3. Use 'Text to Speech' Widget.
